# Hashdump
Dump account data and password hashes from an offline Active Directory database.

```console
go install go.foxforensics.dev/hashdump@latest
```

## Usage
```console
$ hashdump [hv] NTDS SYSTEM
```

### Options
* `-h` Show password history
* `-a` Show detailed infos
